Política de Privacidad

Última actualización: 23 de septiembre de 2025

Este sitio web y el proyecto Modelo Pront son de código abierto y de uso gratuito. No recopilamos, almacenamos ni procesamos ningún dato personal de los visitantes.

¿Qué información recopilamos?

Ninguna. Este proyecto no utiliza cookies, formularios de contacto, analíticas (como Google Analytics), ni ningún otro mecanismo para recolectar información personal o técnica de los usuarios.

Uso del sitio web

Modelo Pront es una herramienta estática que se ejecuta completamente en tu navegador. No envía ni recibe datos de internet. Todo el procesamiento ocurre localmente en tu dispositivo.

Cambios en esta política

Dado que no se recopilan datos, es poco probable que esta política cambie. Si lo hace, se actualizará esta página.

Si tienes dudas, consulta la sección de Contacto.
